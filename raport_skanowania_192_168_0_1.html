Legenda Otwartych Portów

  • 22: SSH (Secure Shell)
  • 53: DNS (Domain Name System)
  • 80: Serwer webowy http
  • 135: Microsoft RPC (Remote Procedure Call)
  • 139: NetBIOS Session Service
  • 443: Serwer webowy https
  • 445: Microsoft-DS (SMB - Server Message Block)
  • 631: IPP (Internet Printing Protocol) - drukarki sieciowe
  • 4357: Home Assistant (HTTP)
  • 5000: Serwery deweloperskie (często Flask w Pythonie)
  • 8000: Synology NAS
  • 8001: Synology NAS
  • 8080: Audiobookshelf (HTTP) - Domyślny, ale konfigurowalny
  • 8081: Alternatywny HTTP (często używany przez serwery proxy lub aplikacje Java)
  • 8096: Jellyfin (HTTP)
  • 8123: Home Assistant (HTTPS)
  • 9090: Prometheus Server
  • 9100: Exporter Prometheus (metryki aplikacji)
  • 13378: Audiobookshelf (HTTP) - Domyślny, ale konfigurowalny
  • 32400: Plex

Legenda Skrótów Systemów/Urządzeń

  • HomeAsst: Home Assistant (wykryto port 8123 lub 4357)
  • Lin/Media (Samba,RDP?): Linux Media Center (wykryto SSH, Samba(139), potencjalnie RDP)
  • NAS/MediaSrv: NAS z usługami multimedialnymi (SSH, Web, SMB, Plex, Jellyfin, etc.)
  • NetDev/Web: Urządzenie sieciowe, urządzenie IoT lub serwer z interfejsem webowym.
  • Nieznany (No Ports): Nieznany typ urządzenia (brak otwartych portów)
  • Printer (IPP,Web): Drukarka sieciowa (wykryto IPP i interfejs web)
  • Win (SMB): System Windows (wykryto porty SMB/NetBIOS/RPC)
